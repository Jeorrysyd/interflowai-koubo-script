# 输入卡

用户没有完整 brief 时，用这张卡补齐。不要机械追问所有字段，缺关键字段时最多问 3 个问题。

```text
我的想法：

目标用户：

这个想法来自哪里：
例如新闻、客户对话、活动现场、工作流实践、产品 Demo、评论区问题

我想让用户相信什么：

我手上的证据或素材：
例如截图、现场照片、案例、数据、对话、文档、表格、工具界面

发布平台：
例如小红书、抖音、视频号、B 站、LinkedIn

希望用户评论什么：

这条内容最好留下什么资产：
例如潜在客户、合作线索、案例、模板、下一条选题、工作坊入口
```

## 最少输入

如果用户只给一句话，至少补齐或推断：

- 目标用户。
- 想让用户相信的主张。
- 可用证据。

不确定时先写假设，再继续生成。
