# 样例：活动复盘

## 输入

```text
我的想法：我参加一个线下 AI 活动后发现，大家不再只问模型能力，而是在问 AI 怎么进入真实业务流程。
目标用户：AI 创业者、品牌运营、企业创新负责人
这个想法来自哪里：线下活动现场、嘉宾分享、观众提问和会后交流。
我想让用户相信什么：AI 活动真正有价值的不是信息量，而是能不能带走一个可执行的业务问题。
我手上的证据或素材：现场照片、议程截图、几个高频问题、会后加微信的合作线索。
希望用户评论什么：你参加 AI 活动最想带走工具、案例，还是业务机会？
这条内容最好留下什么资产：下一场工作坊主题和潜在合作线索。
```

## 期望输出特征

- 主模型：活动复盘型。
- 不按时间线复述活动。
- 第一段先讲现场洞察。
- 中段只保留能证明洞察的细节。
- 结尾说清活动留下了什么资产。
